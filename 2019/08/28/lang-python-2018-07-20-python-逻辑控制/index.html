Python-逻辑控制

条件语句

if 判断条件1.1 and 判断条件1.2`:`
    执行语句1……
elif 判断条件2.1 or 判断条件2.2`:`
    执行语句2……
elif (num >= 0 and num <= 5) or (num >= 10 and num <= 15)`:`
    执行语句3……
else`:` 执行语句4……  #同一行的位置上使用if条件判断语句

循环for

for iterating_var in sequence:
    statements(s)

循环嵌套for

for iterating_var in sequence:
    for iterating_var in sequence:
       statements(s)
    statements(s)

break,continue,pass

# 跳出最近的一层循环
$ break

# 跳过本次循环，继续下一次循环
$ continue

# 占位符，后面的还运行
$ pass
